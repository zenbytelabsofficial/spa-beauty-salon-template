# Brancy - Cosmetic & Beauty Salon Website Template

A premium, fully responsive Cosmetic, Spa & Beauty Salon Website Template. This template is designed for beauty salons, spa centers, cosmetic shops, and hair care businesses.

## Key Features

- **25+ HTML Pages**: Includes multiple homepages, shop layouts, product details, blog layouts, cart, checkout, FAQ, and contact page.
- **Fully Responsive**: Optimized for desktops, tablets, and mobile devices.
- **Modern UI/UX**: Features clean typography, smooth transitions, premium image sliders, and interactive shopping widgets.
- **E-Commerce Ready**: Structured pages for cart, checkout, wishlist, compare, and product variants.

## Getting Started

To view the website locally with all styles, fonts, and assets loaded correctly, run a local web server from the project root:

### Start Local Server
```bash
python -m http.server 8000
```

Then open the following URL in your web browser:
[http://localhost:8000/scraped_site/brancy/brancy/index.html](http://localhost:8000/scraped_site/brancy/brancy/index.html)

---

## Deployment to Vercel

This website is ready to be hosted on Vercel. 

### Configuration Settings
When importing your repository in Vercel, change the **Root Directory** setting:
- **Root Directory**: `scraped_site/brancy/brancy`

This will serve the website directly at the root of your Vercel URL (e.g., `https://your-spa-website.vercel.app/`).
